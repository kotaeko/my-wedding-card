## ✏️ 콘텐츠 입력 (직접 content.js 수정)

### 타임라인 (`timeline` 배열)
- [ ] 첫만남 날짜/설명 실제 내용으로 수정
- [ ] 첫여행 날짜/설명 실제 내용으로 수정
- [ ] 프로포즈 날짜/설명 실제 내용으로 수정
- [ ] 결혼 설명 문구 다듬기
- [ ] 각 이벤트에 사진 넣기 (드라이브 파일 ID 추가 선택)

### 어릴적 사진 (`childhoodPhotos`)
- [ ] 신랑 어릴 적 사진 구글 드라이브에 업로드 후 파일 ID 입력
- [ ] 신부 어릴 적 사진 구글 드라이브에 업로드 후 파일 ID 입력

### 계좌번호 (`accounts`)
- [ ] 신랑 계좌번호 입력 (은행명 + 계좌번호)
- [ ] 신랑 아버지 계좌번호 입력
- [ ] 신랑 어머니 계좌번호 입력
- [ ] 신부 계좌번호 입력
- [ ] 신부 아버지 계좌번호 입력
- [ ] 신부 어머니 계좌번호 입력

### 교통편 (`transportInfo`)
- [ ] 버스 노선 정보 입력
- [ ] 자가용 주차 안내 입력

### 갤러리 (`couplePhotos`)
- [ ] 커플 사진 구글 드라이브에 업로드 후 파일 ID 추가

### OG 이미지 (카카오톡 공유 미리보기)
- [ ] 1200×630 사이즈 이미지 제작 (캔바 등 활용)
- [ ] Cloudinary 업로드 후 `ogImageUrl` 입력
